# 카카오톡 쇼핑하기 클론코딩

## 1. 프로젝트 소개

- 카카오톡 쇼핑하기 클론코딩 프로젝트입니다.

## 2. 프로젝트 실행하는 법

- 프로젝트를 클론합니다.

```bash
git clone [레포지토리 주소]
```

- 프로젝트를 설치합니다.

```bash
npm install
```

- 프로젝트를 실행합니다.

```bash
npm start
```

## 3. 주의사항

해당 레포지토리의 main브랜치는 카카오톡 IDE 크램폴린에 배포하기 위한 것으로 로컬에서 실행 시 오류가 발생할 수 있습니다.
로컬 실행을 위해서는 feat-ParkDongjin 브랜치를 사용하세요
